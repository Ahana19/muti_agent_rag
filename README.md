# Enterprise Multi-Agent RAG Platform

A production-style enterprise AI application built from scratch to simulate a real client project.

## Goal

Develop a secure multi-agent Retrieval-Augmented Generation (RAG) platform capable of:

- answering questions from enterprise documents
- reducing hallucinations
- evaluating AI responses
- monitoring system performance

## Tech Stack

- Python
- FastAPI
- LangChain
- LangGraph
- Ollama
- FAISS
- Docker
- SQLite
